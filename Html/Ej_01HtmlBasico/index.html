Cuerpo principal del contenido

La etiqueta main es en HTML5 una etiqueta semántica. Esto quiere decir que tiene que ver con el significado de lo contenido dentro de ella. Define el contenido principal de la página. Únicamente puede haber una sola etiqueta como ésta en todo el documento. El contenido escrito aquí dentro no se repite en el resto del sitio (todo lo contrario a los encabezados de la página, los menúes de navegación y los pies de página). Es usado por buscadores para indexar. Conviene usar aquí textos que ayuden a encontrar la página luego de procesos de indexación de los buscadores.

Sección etiquetas semánticas estructurales más importantes:

  • HEADER
  • FOOTER
  • SECTION
  • ARTICLE
  • NAV
  • ADRESS

Las etiquetas semánticas estructurales están relacionadas directamente con el significado del contenido y no con los estilos ni las posiciones dentro del documento.

Reglas para etiquetas semánticas

  • Los distintos tipos de contenido se pueden asociar a las etiquetas SECTION
  • Las unidades de contenido se asocian a las etiquetas ARTICLE. O sea que un SECTION anida múltiples ARTICLES.
  • Las etiquetas HEADER y FOOTER pueden ser anidadas dentro de otras HEADER o FOOTER. O sea un FOOTER podría anidar otro HEADER y FOOTER.
  • La etiqueta NAV debe ser única en el documento y corresponde a los elementos de navegación del documento.
  • Reservar etiqueta h1 para usar dentro de etiquetas top level como main, header y footer.
