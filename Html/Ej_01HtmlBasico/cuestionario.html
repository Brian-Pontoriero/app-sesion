Cuestionario


Preguntas:

  1. Dar un par de ejemplos de uso de etiquetas meta.

  2. Nombrar al menos 2 atributos aplicables a la etiqueta HTML

  3. ¿Cómo se relaciona un header http con la etiqueta head de HTML?

  4. ¿Qué es UTF8?

  5. ¿Qué atributo HTML sirve para aplicar estilo, a qué etiqueta es aplicable?

  6. ¿Dónde se indica que un enlace debe ubicar el documento en una nueva pestaña?

  7. Explicar cuando usarían etiquetas semánticas y con que propósito.

  8. ¿Qué atributos de estilo básicos aplicarían a una tabla? Explicar.

  9. ¿Qué diferencia existe entre la etiqueta body de un documento HTML y el body del protocolo HTTP?

  10. ¿Qué atributos marcan la agrupación de celdas de una tabla?. ¿A que etiquetas se aplican?


Respuestas:

  1. Ejemplos de etiquetas meta son, la etiqueta title y meta. Title informa el título de la página en la pestaña de la misma, y en meta se aloja informacion útil para el SEO.

  2. Atributos aplicables a las etiquetas html pueden ser style, controls, widht, height e id entre otros.

  3. Se relacionan, de modo en que la información del protocolo http requerida por los servidores está alojada entre las etiquetas "head" y provee información general (metadatos) acerca del documento, incluyendo su título, enlaces a scripts y hojas de estilos.

  4. UTF8 es el valor del código de lenguaje español.

  5. El atributo que sirve para aplicar estilo es style, y se puede aplicar en cualquier etiqueta html.

  6. Se indica que un enlace debe ubicar el documento en una nueva pestaña en la etiqueta anchor (a), por medio del atributo target ="_blank".

  7. Las etiquetas semánticas se usan para estructurar el contenido del documento HTML. Le dan mejor funcionamiento a la página y agilizan el trabajo del SEO. Se deben usar siempre que se desee crear una página web de calidad.

  8. A una tabla se le debe aplicar mínimamente atributos de background, width, height y border para su correcta visualización. De lo contrario, no es posible hacerlo.

  9. El body del http guarda información compartida opcional, entre el cliente y el servidor. Mientras que el body del documento html constituye el contenido de una página web.

  10. Los atributos de agrupacion de celdas son colspan (agrupación en fila) y rowspan (agrupación en columna). Se aplican a las etiquetas td.
